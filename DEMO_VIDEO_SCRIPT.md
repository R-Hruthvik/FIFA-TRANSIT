## Recording Tips

- Use **1080p 60fps** screen capture (OBS or built-in)
- Keep cursor movements smooth
- Use the **Demo Mode** button (top-right) to cycle through realistic data
- If data is sparse, click "START DEMO" for 60 seconds of scripted data
- Record in one continuous take, edit cuts later
- Add subtle background music (lo-fi or ambient)
